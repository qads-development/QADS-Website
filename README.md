# QADS Website
This is the primary repository for the QADS website. Please note that while our platforms are open-source, they are not directly open for community modifications. You must contact help@qads.cloud to become a contributor. QADS.cloud is not a company and does not hold any copyrights, so you are free to use this code as you see fit. We appreciate attribution.

Developped in HTML. Many components are powered by Tailwind CSS.

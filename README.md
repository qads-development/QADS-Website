# QADS Website
This is the primary repository for the QADS website. Please note that while our platforms are open-source, they are not directly open for community modifications. You must contact help@qads.cloud to become a contributor. This is code is under the MIT license, meaning it is free to use. We appreciate attribution.

Developped in HTML. Many components are powered by Tailwind CSS.
* This is not the tracking software, nor the dashboard.

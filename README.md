# QADS Website
This is the primary repository for the QADS website. Please note that while our platforms are open-source, they are not directly open for community modifications. You must contact help@qads.cloud to become a contributor. This is code is under the AGPL-3.0.

Developped in HTML. Many components are powered by Tailwind CSS.
